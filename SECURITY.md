# Security Policy

## Scope
This repository contains **sanitized governance documentation and evidence artifacts** created for demonstration and educational purposes.

No production credentials, secrets, or live tenant access are included.

---

## Reporting a Security Issue
If you believe you have found:
- accidentally exposed sensitive information
- a sanitization error
- content that should be removed or redacted

Please report it responsibly.

**Contact:**  
Email: security [at] robertmyers.cloud *(or update to your preferred contact)*

---

## Disclosure Policy
- Do not open public issues for sensitive findings.
- Provide sufficient detail to identify the affected file(s).
- Verified issues will be remediated promptly.

---

## Safe Harbor
This repository does not represent production authority over any employer, customer, or third-party tenant.
